LRM¶

Language Reference Manual is the name given to IEEE Standard documents defining Hardware Description Languages:

  • VHDL LRM revisions:

    • 1076-2019.

    • 1076-2008 aka IEC 61691-1-1:2011.

    • 1076-2002 aka IEC 61691-1-1 Ed.1 (2004-10).

    • 1076-2000.

    • 1076-1993.

    • 1076-1987.
